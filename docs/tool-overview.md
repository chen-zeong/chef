# Chef 工具模块与生产力规划

本文档梳理了 Chef 应用当前可用的工具模块，并给出了面向办公与编程日常的扩展建议，帮助规划后续的功能路线。

## 现有模块速览

| 模块 | 现有工具 | 状态 | 主要使用场景 |
| --- | --- | --- | --- |
| JSON 工具 | JSON 解析器（ready） | 已上线 | 前端/后端调试 JSON 数据 |
| 加密 | MD5 摘要（ready）<br>Base64 编解码（ready）<br>AES 对称加密（ready）<br>URL 编解码（ready） | 已上线 | 字符串加解密、接口调试 |
| 图片处理 | 图片压缩（soon）<br>格式转换（soon） | 规划中 | 设计稿与素材的批处理 |
| 电脑操作 | Host 管理（ready）<br>环境变量（ready）<br>电脑常亮（ready）<br>全局文件搜索（ready） | 已上线 | 本机配置与临时调试 |
| 图标 | SVG 预览（ready）<br>图片转图标（ready） | 已上线 | 图标预览、图标生成 |

## 模块详解与补充想法

### JSON 工具
- 聚焦于 JSON 数据的解析、格式化与校验，适合接口联调及日志排查。
- 可进一步加入 `JSON Diff`、`JSON 模板生成` 等辅助工具，提升结构化数据的对比与复用效率。

### 加密
- 包含常见的哈希、对称加密与编码工具，已经覆盖多数流程所需的「快速校验」场景。
- 扩展建议：`JWT 解码`、`HMAC 生成器`、`加盐哈希助手`，以便处理鉴权相关脚本。

### 图片处理
- 已规划的 `图片压缩`、`格式转换` 面向设计与运营场景。
- 建议补充 `批量重命名`、`分辨率调整`、`水印添加` 等功能，完善素材处理链路。

### 电脑操作
- `Host 管理`、`环境变量`、`电脑常亮` 解决本地环境切换、脚本初始化与演示场景需求。
- 新增 `全局文件搜索`，利用 rust_search 在指定目录及附加路径内快速定位文件，支持隐藏文件、体积阈值与创建/修改时间筛选，适合大型仓库或资料盘定位。
- 可进一步加入 `端口占用检测`、`进程优先级调整`、`Clipboard 历史` 等系统效率工具。

### 图标
- `SVG 预览` 与 `图片转图标` 满足图标生成与适配需求，覆盖设计与开发协作流程。
- 建议新增 `图标库管理`（批量上传、分类、搜索）以及 `Favicon 生成器`，一键输出多尺寸产物。

## 建议新增模块（偏办公与编程日常）

### 多媒体生产力
- `视频压缩`：针对内嵌演示、会议纪要的视频快速降码率，支持预设清晰度。
- `音频转码`：会议录音从 WAV/CAF 批量转 MP3/AAC，方便分享与归档。
- `字幕生成`：利用本地模型或第三方接口将音频/视频转字幕，并提供时间轴编辑。

### 办公文档助手
- `PDF 合并/拆分`：整理合同、报告材料的常用操作。
- `PPT 图片压缩`：批量重压演示文稿中的图片资源，缩小文件体积。
- `文档转格式`：Word ↔ PDF、Markdown ↔ HTML 的双向转换。

### 编程效率工具
- `正则表达式调试器`：实时测试正则匹配，提供常用片段模板。
- `Cron 表达式生成器`：可视化配置计划任务时间，输出多语言示例。
- `Git Patch 可视化`：加载补丁文件并渲染差异，便于 Code Review 与分享。

### 协作与记录
- `截图标注`：快速截屏并添加箭头、马赛克，适合需求讨论与缺陷反馈。
- `会议纪要模板`：生成结构化纪要骨架，结合快捷文本片段，提高整理效率。
- `待办清单同步`：与常用任务管理工具（如 Todoist、TickTick）对接，统一记录日常任务。

## 规划建议

- **优先级排序**：结合现有组件和接口储备，先实现多媒体与办公类工具，可与图片模块复用大量代码。
- **模块化复用**：将文件拖拽、批量处理、进度提示等能力抽象为共享组件，降低新增工具的开发成本。
- **跨平台适配**：针对 Tauri 框架，关注文件系统权限与性能瓶颈，必要时采用后台任务或原生插件。
- **可观测性**：在新工具中加入基础的操作日志或导出记录，有助于后续分析使用频次与反馈。
